# Assets - Myweb Dashboard

Carpeta de recursos estaticos organizados por tipo para uso en el tablero Power BI.

## Estructura

```
assets/
|-- icons/        -> Iconos SVG o PNG (KPI, navegacion, botones)
|-- images/       -> Imagenes generales (fotos o ilustraciones)
|-- backgrounds/  -> Fondos de paginas o secciones del reporte
|-- logos/        -> Logotipos de marca, empresa o proveedores
```

## Uso rapido

1. Copia cada archivo en la carpeta segun su tipo.
2. En Power BI Desktop usa Insertar > Imagen para agregar recursos.
3. Para fondos usa Formato > Fondo de pagina.

## Formatos recomendados

- icons: .svg, .png
- images: .png, .jpg
- backgrounds: .png, .jpg
- logos: .svg, .png
